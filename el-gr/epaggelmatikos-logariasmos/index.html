Συχνές ερωτήσεις για Επαγγελματικούς λογαριασμούς

Τι χρειάζομαι για να ανοίξω έναν Επαγγελματικό λογαριασμό;

Πριν μπορέσουμε να ανοίξουμε τον Επαγγελματικό σας λογαριασμό, πρέπει πρώτα να επαληθεύσουμε την ταυτότητά σας. Το μόνο που χρειάζεται να κάνετε είναι να μεταφορτώσετε μια φωτογραφία ενός εγγράφου πιστοποίησης ταυτότητας, όπως ένα έγκυρο δίπλωμα οδήγησης, ένα διαβατήριο ή ένα εθνικό δελτίο ταυτότητας.

Ενδέχεται επίσης να σας ζητήσουμε να προσκομίσετε έναν λογαριασμό υπηρεσιών κοινής ωφέλειας ή ένα αντίγραφο κίνησης τραπεζικού λογαριασμού, ως μέσο επαλήθευσης της διεύθυνσής σας.

Πόσο γρήγορα μπορώ να ανοίξω έναν Επαγγελματικό λογαριασμό;

Για να ανοίξετε έναν Επαγγελματικό λογαριασμό στη SumUp, θα χρειαστείτε συνήθως λιγότερο από 10 λεπτά. Μόλις εγγραφείτε και επαληθεύσετε την ταυτότητά σας, ο IBAN σας είναι έτοιμος να χρησιμοποιηθεί αμέσως, ώστε να μπορείτε να προσθέτετε άμεσα χρήματα και να πραγματοποιήσετε μεταφορές, όσο η Mastercard σας βρίσκεται καθ' οδόν.

Σε σπάνιες περιπτώσεις, μπορεί να μας πάρει λίγο περισσότερο χρόνο για να επαληθεύσουμε τα έγγραφά σας, ειδικά αν η φωτογραφία του εγγράφου πιστοποίησης της ταυτότητάς σας είναι θολή ή αν η διεύθυνση που δηλώσατε δεν ταιριάζει με αυτήν που αναγράφεται στο αποδεικτικό της διεύθυνσης.

Πόσο κοστίζει το άνοιγμα ενός Επαγγελματικού λογαριασμού;

Το άνοιγμα ενός Επαγγελματικού λογαριασμού στη SumUp είναι 100% δωρεάν. Επίσης, δεν χρεώνουμε μηνιαίο ή ετήσιο τέλος ή τέλος για την αποστολή ή τη λήψη χρημάτων.

Είναι πραγματικά ένας δωρεάν Επαγγελματικός λογαριασμός;

Ναι, όταν λέμε "δωρεάν", το εννοούμε. Δεν θα σας κοστίσει τίποτα να ανοίξετε τον Επαγγελματικό σας λογαριασμό, ούτε για να στέλνετε, να λαμβάνετε, να ξοδεύετε ή να διαχειρίζεστε τα χρήματά σας.

Τα μοναδικά τέλη που χρεώνουμε είναι για αναλήψεις που υπερβαίνουν το μηνιαίο επιτρεπόμενο όριο δωρεάν αναλήψεων και για αναλήψεις που πραγματοποιούνται στο εξωτερικό. Και στις δύο περιπτώσεις, το τέλος είναι 2%.

Μπορώ να διαχειρίζομαι τα έξοδά μου;

Ναι, φυσικά. Με τον δωρεάν Επαγγελματικό λογαριασμό SumUp, μπορείτε να διαχειρίζεστε εύκολα τα έξοδά σας και να εξοικονομειίτε πολύτιμο χρόνο για την επιχείρησή σας. Δείτε πώς:

Αυτόματες κατηγορίες

  • Οι συναλλαγές κατηγοριοποιούνται αυτόματα σε εξατομικευμένες κατηγορίες εξόδων

  • Όλα τα βασικά στοιχεία είναι προ-συμπληρωμένα, μειώνοντας την ανάγκη για χειροκίνητες εισαγωγές

Παρακολούθηση εξόδων

  • Παρακολουθείτε εύκολα τα έξοδά σας με αναλύσεις ανά κατηγορία εξόδου

  • Παρακολουθείτε τις πληρωμές ΦΠΑ στα έξοδά σας

Σκανάρετε τις αποδείξεις σας

  • Ανεβάστε φωτογραφίες των αποδείξεών σας και επισυνάψτε τις σε κάθε έξοδο

  • Αποθηκεύστε ηλεκτρονικά της αποδείξεις σας, για να παραμένετε οργανωμένοι

Εύκολη φορολογική προετοιμασία

  • Κάντε λήψη ενός αρχείου CSV με τα έξοδα για φορολογικούς σκοπούς

  • Προσκαλέστε τον λογιστή σας να διαχειρίζεται απευθείας τα τιμολόγια και τα έξοδά σας

Πώς μπορώ να ανανεώσω το υπόλοιπο του λογαριασμού μου;

Μπορείτε να προσθέσετε χρήματα στον Επαγγελματικό λογαριασμό SumUp που διαθέτετε, είτε με τραπεζικό έμβασμα είτε με πάγια εντολή. Μόλις ανανεώσετε το υπόλοιπό σας, μπορείτε να αρχίσετε να χρησιμοποιείτε τον λογαριασμό σας για να πραγματοποιήσετε πληρωμές ή αγορές πριν καν φτάσει η κάρτα σας.

Δείτε πώς μπορείτε να βρείτε τα στοιχεία του λογαριασμού σας, συμπεριλαμβανομένου του IBAN σας:

Στην εφαρμογή

  1. Πατήστε "Λογαριασμός" στην αρχική οθόνη.

  2. Πατήστε το εικονίδιο ρυθμίσεων στην επάνω γωνία.

  3. Επιλέξτε "Στοιχεία λογαριασμού" για να δείτε όλα τα στοιχεία του λογαριασμού σας.

Στον υπολογιστή σας

  1. Μεταβείτε στο Προφίλ SumUp που διαθέτετε και επιλέξτε την καρτέλα "Επαγγελματικός λογαριασμός" στα αριστερά.

  2. Εδώ μπορείτε να βρείτε όλες τις πληροφορίες που χρειάζεστε για να μεταφέρετε χρήματα στον λογαριασμό σας.

Για περισσότερες πληροφορίες, ανατρέξτε στο Κέντρο Υποστήριξης.

Η SumUp είναι τράπεζα;

Όχι, η SumUp δεν είνια τράπεζα. Είμαστε ένας από τους πιο κορυφαίους παρόχους χρηματοπιστωτικών υπηρεσιών και ένα εξουσιοδοτημένο ίδρυμα ηλεκτρονικού χρήματος, το οποίο ρυθμίζεται από την Κεντρική Τράπεζα της Ιρλανδίας (αριθμός αναφοράς: Αριθ. C195030). Αυτό μας επιτρέπει να δημιουργούμε ασφαλείς λογαριασμούς ηλεκτρονικού χρήματος, IBAN, κωδικούς BIC και Mastercards για τη μικρή σας επιχείρηση.

Προστατεύουμε τα χρήματα των πελατών μας σε λογαριασμούς ηλεκτρονικού χρήματος που είναι ξεχωριστοί από τους τραπεζικούς λογαριασμούς της SumUp, γεγονός που σημαίνει ότι τα χρήματά σας προστατεύονται για κάθε ενδεχόμενο. Πιστεύουμε ότι αυτό μας επιτρέπει να προσφέρουμε μια πιο ασφαλή και ευέλικτη εναλλακτική λύση επαγγελματικών τραπεζικών λογαριασμών,, παρέχοντας στους πελάτες μας καλύτερη προστασία από απάτες με λιγότερη γραφειοκρατία.

Πώς προστατεύονται τα χρήματά μου;

Τα χρήματά σας είναι πάντα προστατευμένα.

Ως εξουσιοδοτημένο ίδρυμα ηλεκτρονικού χρήματος στην Ευρώπη, ρυθμιζόμαστε από την Κεντρική Τράπεζα της Ιρλανδίας και λειτουργούμε σύμφωνα με τους κανόνες και τους κανονισμούς της.

Αυτό σημαίνει ότι φυλάσσουμε τα χρήματά σας σε ξεχωριστούς λογαριασμούς που δεν είναι συνδεδεμένοι με τους δικούς μας, ώστε, σε κάθε περίπτωση, τα χρήματά σας να είναι ασφαλή και διαθέσιμα σε εσάς.

Επίσης, χρησιμοποιούμε την τεχνολογία ασφαλείας 3D για να εγγυηθούμε την ασφάλεια των πληρωμών σας. Κάθε φορά που πραγματοποιείτε μια online πληρωμή με τη SumUp Mastercard, θα λαμβάνετε μια ειδοποίηση στο τηλέφωνό σας που θα σας ζητά να επαληθεύσετε τη συναλλαγή.
