Le paiement sans contact en toute simplicité

Comment configurer un code QR

Créer

Allez dans la section Codes QR dans l'onglet « Plus » de l'application SumUp. Appuyez sur « Créer un code QR ».

Saisissez les informations

Donnez un nom et une description à votre code QR. Appuyez sur « Enregistrer ».

Partager ou imprimer

Sélectionnez le code QR que vous venez de créer et choisissez de le partager ou de l'imprimer.

Scanner pour payer.

Scanner

Votre client utilise son téléphone pour scanner le code QR.

Payer

Un site web s'ouvre sur lequel votre client saisit le montant à payer et les informations de sa carte.

Confirmer

Le client vous montre l'écran affichant la confirmation du paiement pour valider ce dernier. Vous pouvez également vérifier le paiement dans l'historique des ventes de votre application SumUp. 

Téléchargez l'application pour commencer

Autres questions et réponses

Comment puis-je imprimer un code QR pour l'afficher ?

  1. Dans la liste des codes QR que vous avez créés, cliquez sur le code QR que vous souhaitez imprimer et choisissez « Code QR pour affichage en boutique ».

  2. Vous verrez un aperçu de votre affichage au format PDF. Cliquez sur le bouton de partage pour imprimer l'affichage sur une imprimante connectée à votre téléphone ou pour envoyer l'affichage par e-mail pour l'imprimer depuis un ordinateur.

Puis-je inclure un code QR dans un document ou un affichage que j'ai moi-même créé ?

Bien sûr ! 

  1. Dans la liste des codes QR que vous avez créés, cliquez sur le code que vous souhaitez utiliser.

  2. Dans la liste qui s'affiche, cliquez sur « Code QR ».

  3. Vous pouvez maintenant envoyer le fichier vers l'appareil que vous allez utiliser pour créer votre affichage.

Que voit mon client lorsqu'il scanne un code QR ?

Quand un client scanne un code QR, une page de paiement s'affiche depuis laquelle le client saisit le montant et confirme le paiement. Enfin, le client vous montre l'écran de son smartphone affichant le montant payé et l'heure actuelle afin de confirmer le paiement.

Quels sont les moyens de paiement acceptés via code QR ?

Vous pouvez accepter toutes les cartes de crédit ou de débit affichant le logo Visa, Mastercard, American Express, Discover, Diners Club, ou Union Pay. Nous ajouterons d'autres moyens de paiement à l'avenir.

Puis-je utiliser un code QR plusieurs fois ?

Vous pouvez recevoir plusieurs paiements via un seul code QR. Les codes QR sont actifs jusqu'à ce que vous les supprimiez de la liste des codes QR que vous avez créés. Vous pouvez également utiliser le même code QR à plusieurs endroits dans votre entreprise. Par exemple, sur chaque table ou chaque chaise, sur votre comptoir, dans votre vitrine ou affiché à l'extérieur de votre magasin si vos clients font la queue. Il n'y a aucune limite à votre créativité !

Comment puis-je envoyer un reçu à mes clients quand ils paient avec un code QR ?

Pour envoyer un reçu de paiement à vos clients, ouvrez l'historique des ventes sur votre application SumUp. Sélectionnez la transaction correspondante et cliquez pour afficher tout le menu. Vous pouvez alors envoyer le reçu à votre client par e-mail ou SMS.
