¿Tienes preguntas? Te las respondemos.

¿Qué es un POS o Punto de Venta?

Un punto de venta, también conocido como sistema punto de venta o sistema POS es una herramienta que permite a los negocios gestionar eficientemente sus transacciones y actividades de ventas.

Se compone de un hardware para procesar pagos y un software punto de venta que se ejecuta en ese sistema. Un sistema POS como el Punto de Venta lite permite realizar tareas administrativas como el análisis de informes de transacciones, gestión de sus productos y precios desde una única plataforma.

¿Necesito un lector para aceptar pagos con tarjeta con el punto de venta?

Sí, necesitas un lector de tarjetas SumUp para aceptar pagos con tarjeta u otros medios de pago. El Punto de venta lite es compatible con nuestros lectores Solo y Air. Si aún no tienes uno, puedes conseguirlo acá.

Si ya tienes uno, puedes adquirir el Punto de Venta lite por separado aquí o bien comenzar a usar todas las funciones de nuestros software gratuito en tu propia tablet.

¿Puedo conectar el Punto de Venta a mi perfil SumUp actual?

Sí, si ya tienes un perfil SumUp, solo tienes que iniciar sesión con tus credenciales cuando enciendas tu Punto de Venta Lite.

Una vez que inicies sesión, tendrás acceso inmediato a tu perfil SumUp, tu catálogo de productos y tu historial de ventas.

¿Cuánto cuesta el Punto de Venta lite? ¿Tiene costos mensuales?

A diferencia de la mayoría de los sistemas POS, el Punto de Venta Lite no tiene costo de suscripción ni mensualidad. Basta con adquirir el hardware por $349.000 y podrás utilizar todas sus funciones sin cargos adicionales.

En cuanto a los costos por transacción, pagas la misma comisión de tu lector de tarjetas. Conoce más de nuestras tarifas aquí.

¿El Punto de Venta Lite se conecta con el Lector Solo + Impresora?

Sí, el Punto de Venta Lite se conecta con nuestra amplia variedad de lectores de tarjeta: SumUp Air, SumUp Solo y SumUp Solo + impresora.

¿Puedo conectar mi Punto de Venta lite a una pistola para escanear código de barra?

Sí, puedes conectar un scanner de código de barras para encontrar tus productos más rápido y mejorar el proceso de pago. Para hacerlo necesitarás que tenga un cable Usb tipo C o que se conecte de forma inalámbrica a través de bluetooth. Si tienes dudas sobre los modelos compatibles, contáctanos.
